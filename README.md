# Hotel Reservation System - Java

## Description
A Java-based Hotel Reservation System created as part of the CodeAlpha Internship.
The system allows users to search, book, cancel rooms, simulate payments,
and store bookings using File I/O.

## Features
- Room categorization (Standard, Deluxe, Suite)
- Room booking & cancellation
- Payment simulation
- Booking details view
- File-based storage
- OOP design

## How to Run
1. Compile:
   javac HotelReservationSystem.java
2. Run:
   java HotelReservationSystem
